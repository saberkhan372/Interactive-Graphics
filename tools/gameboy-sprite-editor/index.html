Unit 5 — Game Dev Bridge

Game Boy Palette Sprite Editor

Design a 16 x 16 sprite with four colors. The constraint is strict enough to matter and expressive enough to become a character.

The sprite editor returns to the course's first principle: constraint is not only a limitation, but also a design method. At 16 by 16 pixels, every pixel has to justify its place in the silhouette.
