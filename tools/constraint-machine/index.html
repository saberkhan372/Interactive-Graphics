Unit 0 — History and Constraint

Constraint Machine

Draw the same simple scene across four graphics eras. The subject stays familiar; the palette, resolution, and rules change.

The point is not to make the most polished image. The point is to hold the subject constant while the display system changes, so that palette, resolution, and pixel density become visible as historical forces rather than background facts.
