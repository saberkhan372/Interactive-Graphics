Unit 1 · Lesson 1.1

More BASICs

This lesson moves from the Apple II as hardware to BASIC as a language for sequencing visual action. Line numbers, loops, color, and horizontal lines are enough to make the machine draw with intention.

Source deck: 1.1 - More Basics.pptx · 11 slides

What the class did

The warm-up asked what information would be needed to recreate a student design on an Apple II. The lesson then named BASIC as Beginners' All-purpose Symbolic Instruction Code, introduced its Dartmouth origin, and connected Applesoft BASIC to Microsoft BASIC and the Apple II.

Students studied a short Steve Wozniak Apple II demonstration program and looked up the commands it used: FOR, COLOR, and HLIN. The drawing becomes legible when the loop, color selection, and line command are read together.

Code shown

10 FOR Y=0 TO 15: COLOR=Y
20 HLIN 20-Y, 39 AT Y+Y
30 HLIN 20-Y, 39 AT Y+Y+1
40 NEXT Y

Key idea

BASIC makes repetition visible. A loop does not only save work; it creates a visual structure by changing one variable and drawing again.
