Unit 1 · Lesson 1.2

First Project

The first project asks students to make a logo with a deliberately small vocabulary. Design becomes a problem of choosing which commands are worth using and which pixels are worth spending.

Source deck: 1.2 - First Project.pptx · 8 slides

What the class did

The warm-up asked what students wanted to make with Apple II graphics and how the class might design a logo. The assignment asked for an original Interactive Graphics logo in Applesoft BASIC, submitted as code and a screenshot from the emulator.

The lesson then introduced the core lo-res commands: PRINT, GR, COLOR, PLOT, HLIN, VLIN, TEXT, and REM. Students practiced placing points and drawing horizontal and vertical lines before looking at a more interactive point-motion program.

Code shown

GR
COLOR = 3
PLOT 0, 0
PLOT 0, 39
HLIN 0, 39 AT 20
VLIN 0, 39 AT 20

Key idea

A constraint becomes useful when it forces a hierarchy. A logo on a 40 by 40 grid cannot keep every detail, so the student has to decide which features carry the identity of the design.
