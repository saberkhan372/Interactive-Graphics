Unit 2 · Lesson 2.2

Making New 3D Shapes

The lesson treats a complex shape as a system of smaller decisions. Size and position can be adjusted one cube at a time, and a larger object can be composed from nodes, edges, and repeated parts.

Source deck: 2.2 - Making new 3D shapes.pptx · 4 slides

What the class did

The warm-up asked how the class had created a "3D" cube with code and how the cube could rotate on the x, y, and z axes. The lesson then named the next goal: adjusting the size and position of a cube, then building larger and more complex shapes.

The important shift is from primitive to composition. A student does not need a single command for every possible object. A shape can be built from smaller objects, and those objects can be organized by position, scale, and connection.

Code shown

const parts = [
  { x: 0, y: 0, z: 0, w: 120, h: 40, d: 40 },
  { x: -80, y: 50, z: 0, w: 40, h: 100, d: 40 },
  { x: 80, y: 50, z: 0, w: 40, h: 100, d: 40 }
];

parts.forEach(function(p) {
  push();
  translate(p.x, p.y, p.z);
  box(p.w, p.h, p.d);
  pop();
});

Key idea

A 3D model can be a collection of parts. Composition lets students build forms that are more interesting than a primitive without leaving behind the logic they already understand.

Try it

Open Flatten/Extrude and focus on the side faces. Each side exists because two matching edges are connected. That is the beginning of thinking in nodes and edges.
