Unit 4 · Lesson 4.2

WebGL in p5.js

This lesson names the systems that a full 3D scene will need. Geometry is only the beginning; students will also have to understand light, material, texture, camera, models, and custom shapes.

Source deck: IG 3.2 - WebGl in p5.js.pptx · 9 slides

What the class did

The class discussed how HTML pages can be made more presentable with CSS and JavaScript, then moved into the WebGL sequence. The deck listed the topics that would define the unit: 3D geometries, light and material, texture, camera and perspective, createGraphics, loading OBJ models, and custom 3D shapes.

The portfolio thread stayed present because the WebGL project was not only a sketch to run. It was a project that would eventually need to be shown, explained, and linked from the student's site.

Code shown

function setup() {
  createCanvas(500, 500, WEBGL);
}

function draw() {
  background(235);
  orbitControl();
  box(120);
}

Key idea

A 3D scene is a composition of systems. The lesson sequence isolates those systems before the final project asks students to combine them.
