Unit 4 · Lesson 4.10

WebGL Scene Project

The final WebGL lesson is a composition problem. Lighting, materials, texture, camera, custom geometry, motion, and interaction have been isolated; the project asks students to make them work together in one scene.

Source deck: IG 3.10 - WebGL Project.pptx · 4 slides

What the class did

The deck treated the class period as project time for the WebGL scene. Students were preparing a scene in p5.js and gathering resources for the parts they wanted to push: depth of field, easier camera control, and orbit interaction.

At this point the course turns from demonstration to synthesis. A successful scene needs more than a checklist of functions; it needs a visual idea strong enough to decide which technical systems matter.

Key idea

A scene is a coordinated system. Each feature should make the scene more readable, more expressive, or more interactive; otherwise it is only a command that happens to be present.

Try it

Use the 3D Scene assignment as the project brief, and use Light & Material Lab and Vertex Sculptor as isolation tools when one part of the scene stops making sense.
