Unit 4 · Lesson 4.0

Semester Recap and Portfolio

The second semester begins by turning completed work into an archive students can own. A portfolio site is not decoration around the course; it is the place where code, image, reflection, and revision become visible together.

Source deck: IG 3.0 - Semester 1 recap + Semester 2 preview.pptx · 8 slides

What the class did

The class recapped the first semester: introduction, history, 2D graphics, the move from 2D to 3D, and the projects already completed. The second-semester preview named a larger arc of scene development, game development, hardware, and modern rendering.

The practical project was a personal website for the class. Students created a landing page and project pages for existing and upcoming work, with embedded images, video, code, and short written reflection.

Key idea

A portfolio site makes the course cumulative. Each project remains available as evidence of a technical idea, a visual decision, and a student reflection on what changed while making it.
