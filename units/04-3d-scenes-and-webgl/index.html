Unit 4

3D Scenes and WebGL

This unit treats a 3D scene as a collection of separate systems that have to be understood before they can be composed. Light, material, texture, camera, perspective, and custom geometry each get isolated before students are asked to make them work together.

The biggest unit in the course by lesson count. Every piece of a real 3D scene gets its own lesson: GitHub Pages publishing, custom 3D shapes, lighting and materials, textures, camera and perspective. Students then assemble a full scene assignment that requires nearly all of it at once: translate/rotate, a custom shape built with beginShape/vertex/endShape, texture mapping, at least one light type, motion, and user interactivity.

Covers: semester recap and portfolio publishing, WebGL in p5.js, light and materials, textures, camera and perspective, projection vocabulary, custom 3D shapes, and the WebGL scene project. The original 2022–2023 folder also included a GB Studio branch; this archive separates that branch into Unit 5.

Lesson 4.0

Semester Recap and Portfolio

The second semester opens by turning first-semester work into a public portfolio and setting up the next scene-building arc.
Lesson 4.1

GitHub Pages

Students publish a hand-built site that can hold code, images, video, and reflection for every project in the course.
Lesson 4.2

WebGL in p5.js

The WebGL sequence is named as a set of systems: geometry, light, material, texture, camera, models, and custom shapes.
Lesson 4.4

Light and Materials

Lighting and surface response are separated so students can see which command changes the object and which changes the scene.
Lesson 4.6

Textures

An image becomes a surface treatment, with preload, loadImage, texture, and camera capture as possible inputs.
Lesson 4.7

Camera and Perspective

Students control the viewer as part of the scene, using camera, perspective, orthographic projection, pan, tilt, and lookAt.
Lesson 4.8

Dimensions and Perspectives

Projection vocabulary gives names to the ways a three-dimensional scene can be flattened for a screen.
Lesson 4.9

Custom 3D Shapes

Students move from built-in primitives to geometry made from explicit vertices, faces, and texture coordinates.
Lesson 4.10

WebGL Scene Project

The isolated systems come back together in a scene assignment that requires light, texture, camera, custom geometry, motion, and interaction.
