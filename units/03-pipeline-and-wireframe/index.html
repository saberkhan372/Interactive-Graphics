Unit 3

Pipeline and Wireframe 2023–24 addition

This unit names the hidden machinery that WebGL usually hides. A cube is measured in model space, moved into world space, remeasured by the camera, projected, and finally placed on the screen. Each step is a change in the numbers before it is a change in the image.

This is the unit where the course stops using WebGL and starts explaining it. Students build a 3D wireframe renderer from close to scratch, walking through the actual graphics pipeline: model space, world space, camera space, projection, screen space. The goal is to treat those transformations as real math rather than work the library silently performs. It is a harder, more ambitious unit than the year before, and it now has a dedicated tool that makes each transformation visible.

Covers: 3D wireframes without WebGL, and the underlying 3D pipeline.
