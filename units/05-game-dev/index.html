Unit 5

Game Dev Bridge

The GB Studio branch belongs here because it returns the course to the central idea of constraint. A sprite is still a grid, a palette is still a limit, and the visual problem is still how a small number of choices can become readable and expressive.

Students leave p5.js for GB Studio, a dedicated no-code game engine, but the pixel-art and constraint-thinking skills from Unit 0 carry over directly. A classic Game Boy palette is still just four colors; a sprite is still a small grid of choices. This bridge tool exists so students enter GB Studio already fluent in the constraint.

Covers: From GB Studio to Game Boy, Game Boy Studio, and the "Scene for Game Boy" assignment.
