Assignment · Unit 4

3D Simulation with WebGL

Create a real-time simulation of particles that move and collide in 3D space using WebGL in p5.js. Where the 3D Scene project was about composition, this one is about behavior: velocity, momentum, and elastic collisions, running live.

As assigned 2023–24 · due Friday, April 26 · submit a canvas recording, a GIF, and a project link.

Tools and concepts

  • WebGL in p5.js for rendering 3D graphics;
  • Physics — basic mechanics: velocity, momentum, elastic collisions;
  • Coding — JavaScript and p5.js functions.

Requirements

  • Pick a theme for the simulation;
  • Use 3D shapes to simulate particles within the canvas;
  • Implement collision detection and response — among particles, and with canvas boundaries;
  • Use mouse or keyboard input to add particles into the simulation;
  • Ensure particles move and collide realistically;
  • Use saveGif() to record a GIF demonstrating interaction;
  • The simulation is visually centered on the canvas;
  • The code is well commented, including links to any resources used;
  • The code is well organized, with proper indentation and spacing;
  • Submit GIF and link.

Bonus

  • Implement different particle types with varying sizes or colors;
  • Add lighting effects to enhance the visual quality of the simulation;
  • Include controls to adjust particle speed or the elasticity of collisions.

How it ran

  • Paper design — sketch the particle interactions and how you plan to structure the code;
  • Rough draft — a basic version where creation, movement, and collisions all function;
  • Final draft — refine the code, add the required features, meet all submission criteria.
