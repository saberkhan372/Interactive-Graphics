Assignment · Unit 2

Converting BASIC to p5.js

Take your Animating History project — a program written in Applesoft BASIC — and convert it into a modern web project using p5.js. Same idea, forty years apart. The point is to feel the difference between procedural programming on a 1977 machine and contemporary JavaScript built for creative expression on the web.

As assigned 2023–24 · due February 13.

Background, as the brief put it

Applesoft BASIC was an early programming language for Apple II computers, widely used in education during the 1980s. p5.js is a JavaScript library that makes coding accessible for artists, designers, educators, and beginners, emphasizing visualization and creative expression through the web.

Requirements

  • Research and citations — all outside sources identified in the code via comments;
  • Explanation and documentation — sections of code explained via comments;
  • Code and implementation — the code works as expected;
  • Animation and visual presentation — there is meaningful animation;
  • Organization and submission — link to the p5.js code, with appropriate comments, submitted on time.
