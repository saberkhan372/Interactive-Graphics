Portfolio Site

Build a portfolio website to present your work. Every later project in the course — the 3D scene, the Game Boy scene — ends with "add it to your portfolio," so this site becomes the place where a semester of work accumulates in public. Hand-built HTML and CSS, hosted on GitHub Pages: the same way this course site itself is published.

As assigned 2022–23 · due Wednesday, January 11 · submit a screenshot and a link.

Requirements

  • Create a landing page that explains the goals of the project;
  • The landing page links to each project page;
  • Each project page has samples of the work — pictures or videos — and links;
  • Each project page links back to the landing page;
  • Use CSS and HTML to make the website individualized and presentable;
  • Submit screenshot and link.

Bonus

  • Use a framework like Bootstrap or Jekyll;
  • Use JavaScript to make your website more interactive.

Resources

  • Khan Academy — hosting your website on GitHub
  • Free domain and web hosting with the GitHub Student Developer Pack
