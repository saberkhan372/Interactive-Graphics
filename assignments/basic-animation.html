Assignment · Unit 1 · Final Project

BASIC Animation

Create an animated two-dimensional graphic using BASIC in HGR mode for the Apple II. This is the Unit 1 final: everything the unit taught — loops, sub-routines, user input, color — lands in one running program on a 1977 machine.

As assigned 2022–23 · due in class Monday, November 7.

Requirements

  • Use HGR mode;
  • Prompt the user for information that impacts the animation;
  • Use one or more sub-routine;
  • Use comments to explain code, along with citing references;
  • Use looping;
  • Use multiple colors;
  • Older pixels are removed;
  • Animation restarts once a limit has been reached;
  • Animation fills the screen;
  • Submit screen recording and link to doc with code.

Optional

  • Use an array to store information;
  • Implement collision between objects;
  • Implement leaving the screen on one side and coming out another;
  • Use sound;
  • Use random;
  • Use trigonometry functions.
